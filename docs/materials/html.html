# HTML语义化标签

定义： 指正确的标签包含了正确的内容，结构良好，便于阅读

常见内容： header、footer、main、nav、aside、article等等

# iframe

定义： 会创建一个包含另一个文档的内联框架

提示： 可以将提示文字放在 <iframe></iframe> 之间，在不支持的情况进行提示

缺点： 阻塞主页面的onload事件 不利于SEO 影响页面性能
