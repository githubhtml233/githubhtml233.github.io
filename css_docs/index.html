      # CSS盒子模型

      定义：用来装页面元素的矩形区域，包含内容、边框和内外边距 种类：

      • 标准盒子模型：长宽只包括内容，内外边距和边框会撑大盒子
      • IE盒子模型： 长宽包括内外边距、边框和内容，内外边距和边框不会撑大盒子

      # 画一条0.5px的线

      • 使用meta viewport ,设置initial-scale、min-scale、max-scale为0.5
      • 使用transform:scaleY(0.5)
